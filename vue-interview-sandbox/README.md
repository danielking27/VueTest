# Vue Interview Sandbox

This app is used for live Vue developer interviews.

## Start
cd vue-interview-sandbox

cd vue-interview-sandbox

npm install

npm run dev

## Tasks

- Task 1: Reactive Search Filter
- Task 2: Parent/Child Task List
- Task 3: API Loading and Error State
- Task 4: Bug Fixing
- Task 5: Cart Summary
- Task 6: Login Form Validation
- Task 7: Refactor Discussion

## Candidate Expectations

You are encouraged to think out loud.

Clean, understandable code is more important than speed.

You may ask clarifying questions.

You may use Vue documentation, but avoid using AI tools during the live task unless the interviewer allows it.

## Evaluation Areas

- Vue reactivity
- Component structure
- Props and emits
- Async handling
- Debugging
- Code clarity
- Reasoning and communicationß
